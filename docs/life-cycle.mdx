## Basic description

When you run: `latextomd -i input.tex - o output.tex`: 

1. Pre-Pandoc processing: process enumerate, process tikz/pstricks, some other staff...
2. Convert with pandoc to markdown
3. Post-Pandoc processing: process math mode, some other staff...

## Pre-Pandoc

A class of `LatexToMd`is instentiate whith the `content` of `input.tex`.
