title: CLI
desciption: latextomd command line interface
---

## -h

`latextomd -h`

show help message and exit

## -v

`latextomd -v`

show current version and exit

## -i

`latextomd -i document.tex`

input file path . Produce `document.md`

## -o

`latextomd -i input.tex -o output.md`

output file path.
